# 🌙 月影村 - 多人狼人杀

一个运行在浏览器中的多人狼人杀游戏，适合 6～12 位朋友围坐在一起，用同一台电脑轮流操作。

## 🎮 怎么玩

1. 双击打开 `index.html`，或者拖到浏览器里
2. 输入每位玩家的名字（每行一个）
3. 点击"开始游戏"，大家轮流看自己的身份
4. 天黑请闭眼，按照提示行动吧！

## ✨ 特色

- 🎨 暗色主题 + 月亮星空背景
- 🔊 支持语音播报（浏览器 TTS）
- 📱 自适应布局，手机也能玩
- 🎭 18 种不同角色，每局随机分配
- 💑 支持情侣、混血儿等第三方阵营
