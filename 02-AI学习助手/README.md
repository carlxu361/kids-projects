# 🎓 AI学习助手 - 我的成长之旅

一个专为小学生设计的AI驱动学习助手，帮助制定学习计划、跟踪进度、解锁成就！

## ✨ 主要功能

### 📊 学习仪表盘
- 今日任务概览
- 连续学习天数
- 本周经验统计
- 学习日历热力图

### 📚 学科管理
- 预设5门主科（语文、数学、英语、科学、道德与法治）
- 可添加自定义学科
- 记录当前学习单元

### 📋 AI学习计划
- 选择学科后，AI自动生成本周学习计划
- 每天1-2个学习任务
- 任务包含时长和经验值

### 📈 进度跟踪
- 每日打卡
- 任务完成标记
- 各科进度可视化

### 🏆 成就系统
- 15个成就徽章等你解锁
- 连续打卡、经验积累、全科学习等挑战
- 升级动画和撒花特效

### 🤖 AI助手
- 随时问学科问题
- 请求解释难懂的知识点
- 生成练习题

## 🚀 如何使用

1. 直接用浏览器打开 `index.html`
2. 首次使用会弹出欢迎引导
3. 前往 **设置** 页面配置AI API密钥
4. 开始使用各项功能！

## ⚙️ AI配置

支持两种AI服务：

### OpenAI (ChatGPT)
- 服务：选择 "OpenAI (ChatGPT)"
- API密钥：填入你的OpenAI API Key
- API地址：默认使用官方地址，可自定义

### Claude (Anthropic)
- 服务：选择 "Claude (Anthropic)"
- API密钥：填入你的Anthropic API Key
- API地址：默认使用官方地址，可自定义

> 💡 API密钥仅保存在本地浏览器中，不会上传到任何服务器。
